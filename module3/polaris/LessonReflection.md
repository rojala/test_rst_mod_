# Lesson Reflection

This lesson explores using Rust for data analysis in Jupyter notebooks. It introduces the evcxr_jupyter kernel that allows executing Rust code in notebooks. Key features include displaying custom Rust types as HTML, integrating with libraries like Plotly, and Performance gains over Python.

## Top 3 Key Points

* Evcxr_jupyter enables using Rust in Jupyter for data science workflows
* Rust crates like Polars offer faster DataFrame manipulation than pandas
* Notebooks can render HTML output from custom Rust types

## Reflection Questions

* How could rendering custom Rust types as HTML be useful in notebooks?

* What challenges do you anticipate working with Rust in notebooks?

* What libraries seem most interesting for visualizations or machine learning?

* What use cases are best suited for starting to use Rust and Polars?
